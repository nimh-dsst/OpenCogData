    Contributing to OpenData

    Contributions to OpenData are welcome! All of the code is managed through the GitHub repository.

    How to submit a dataset

    There are three main ways you can submit a dataset for entry to OpenData, which are described in turn below. Note that they are ranked in order of preference, from most preferred to least preferred, by the maintainers of Open Data.

    1. Fill out the Google form

    Have a dataset to submit? Fill out this form: https://forms.gle/N87Dqhs73vbfoPec8

    2. Submit a pull request

    To directly add a dataset yourself, you can open a pull request on Github. First you should fork the OpenData library via GitHub and make modifications on your fork. The OpenData database is written in markdown using a standardized, easy-to-copy template. Once your modification is complete, submit a pull request to merge your changes into the docs branch of OpenData. Pull requests will be reviewed by the maintainers.

    3. Open an issue

    To advertise a dataset for entry into the database, you can open a new issue via GitHub. Please provide a link to a repository for the data and, if available, a link to a corresponding manuscript describing the data.
